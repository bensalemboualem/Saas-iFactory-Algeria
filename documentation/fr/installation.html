Installation IAFactory Algeria

Guide complet pour installer et configurer IAFactory Algeria sur votre système.

Application Web (Aucune installation)

La façon la plus simple d'utiliser IAFactory Algeria est via notre application web.

Accès Immédiat

Rendez-vous sur app.iafactory-algeria.com

  • Aucune installation requise
  • Compatible tous navigateurs (Chrome, Firefox, Safari, Edge)
  • Accessible depuis PC, Mac, tablette, mobile
  • Mises à jour automatiques

Application Desktop (Windows/Mac/Linux)

Windows

  1. Téléchargez IAFactory-Algeria-Setup.exe
  2. Double-cliquez sur le fichier
  3. Suivez l'assistant d'installation
  4. Lancez IAFactory Algeria depuis le menu Démarrer
Configuration minimale Windows :
  • Windows 10 ou supérieur (64-bit)
  • 4 GB RAM (8 GB recommandé)
  • 500 MB d'espace disque
  • Connexion Internet

macOS

  1. Téléchargez IAFactory-Algeria.dmg
  2. Ouvrez le fichier DMG
  3. Glissez IAFactory dans Applications
  4. Lancez depuis Launchpad
macOS Gatekeeper :

Si vous voyez "Application non vérifiée" :

  1. Clic droit → Ouvrir
  2. Ou : Préférences Système → Sécurité → "Ouvrir quand même"

Linux

Debian/Ubuntu :

wget https://downloads.iafactory-algeria.com/iafactory-algeria.deb
sudo dpkg -i iafactory-algeria.deb
sudo apt-get install -f

Fedora/RHEL :

wget https://downloads.iafactory-algeria.com/iafactory-algeria.rpm
sudo rpm -i iafactory-algeria.rpm

AppImage (toutes distributions) :

wget https://downloads.iafactory-algeria.com/IAFactory-Algeria.AppImage
chmod +x IAFactory-Algeria.AppImage
./IAFactory-Algeria.AppImage

Installation Docker

Pour les développeurs et déploiements serveur :

Docker Compose (Recommandé)

# docker-compose.yml
version: '3.8'

services:
  iafactory:
    image: iafactory-algeria/platform:latest
    ports:
      - "8080:8080"
    environment:
      - OPENAI_API_KEY=${OPENAI_API_KEY}
      - ANTHROPIC_API_KEY=${ANTHROPIC_API_KEY}
    volumes:
      - ./data:/app/data
    restart: unless-stopped

Lancer :

docker-compose up -d

Docker Simple

docker run -d \
  -p 8080:8080 \
  -e OPENAI_API_KEY=your_key \
  -v $(pwd)/data:/app/data \
  --name iafactory \
  iafactory-algeria/platform:latest

Installation CLI

Pour les power users et automatisations :

Via npm

npm install -g @iafactory-algeria/cli

Via pip

pip install iafactory-algeria

Vérifier l'installation

iafactory --version
iafactory auth login

Installation SDK/API

Python

pip install iafactory-algeria-sdk

Exemple d'utilisation :

from iafactory import IAFactory

client = IAFactory(api_key="your_api_key")

response = client.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)

JavaScript/TypeScript

npm install @iafactory-algeria/sdk

Exemple :

import { IAFactory } from '@iafactory-algeria/sdk';

const client = new IAFactory({
  apiKey: 'your_api_key'
});

const response = await client.chat.completions.create({
  model: 'gpt-4',
  messages: [{ role: 'user', content: 'Hello!' }]
});

console.log(response.choices[0].message.content);

Configuration Initiale

Première Connexion

  1. Créer un compte
    • Email + mot de passe
    • Ou connexion Google/GitHub
  2. Choisir votre plan
    • Starter (gratuit)
    • Pro (49€/mois)
    • Enterprise (sur devis)
  3. Configurer vos clés API (optionnel)
    • OpenAI, Anthropic, Google...
    • Ou utiliser les clés IAFactory (inclus)

Fichier de Configuration

Créez ~/.iafactory/config.yaml :

# Configuration IAFactory Algeria
api:
  url: https://api.iafactory-algeria.com/v1
  key: your_api_key_here

defaults:
  model: gpt-4
  temperature: 0.7
  max_tokens: 2048

providers:
  openai:
    api_key: sk-...
  anthropic:
    api_key: sk-ant-...

cache:
  enabled: true
  ttl: 3600

logging:
  level: info
  file: ~/.iafactory/logs/app.log

Dépannage

Problème : "Impossible de se connecter"

Solutions :

  1. Vérifiez votre connexion Internet
  2. Désactivez temporairement proxy/VPN
  3. Vérifiez le firewall (autoriser port 443)
  4. Testez : ping api.iafactory-algeria.com

Problème : "Clé API invalide"

Solutions :

  1. Vérifiez que la clé est correctement copiée (pas d'espaces)
  2. Regénérez une nouvelle clé depuis le dashboard
  3. Vérifiez que votre abonnement est actif

Problème : "Installation bloquée par antivirus"

Solutions :

  1. Ajoutez IAFactory à la liste blanche de votre antivirus
  2. Téléchargez depuis le site officiel uniquement
  3. Vérifiez la signature numérique du fichier
