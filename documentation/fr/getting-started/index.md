# Premiers pas sur IAFactory

Bienvenue sur IAFactory Algeria ! Cette page vous guide pour démarrer rapidement sur la plateforme, que vous soyez utilisateur ou développeur.

## 1. Créer un compte
- Rendez-vous sur [iafactoryalgeria.com](https://www.iafactoryalgeria.com) et inscrivez-vous.
- Accédez à votre espace personnel après connexion.

## 2. Générer une clé API (option développeur)
- Depuis le portail développeur, créez une clé API pour accéder aux services IAFactory.

## 3. Première session dans le chat
- Choisissez un assistant ou un modèle IA (ex : juridique, fiscal, gpt-4.1…)
- Posez votre première question dans l’interface chat.

## 4. Utiliser le reprompt / changer d’assistant
- Relancez une question avec un autre modèle pour comparer les réponses.
- Changez d’assistant selon votre besoin métier.

## 5. Créer un assistant simple
- Depuis l’interface, créez un assistant personnalisé (ex : Fiscalité PME).
- Paramétrez les instructions et le secteur d’activité.

## Pour les développeurs
- Consultez la [documentation API quick-start](../api/quick-start.md) pour intégrer IAFactory dans vos applications.

Pour toute aide, contactez support@iafactory.dz.
