# Présentation IAFactory

IAFactory est une plateforme d’intelligence artificielle souveraine, conçue pour l’Algérie et la Suisse. Elle propose un accès unifié à plusieurs modèles IA de pointe (gpt-4.1, claude, gemini, mistral, deepseek…) et des assistants sectoriels adaptés aux besoins métiers : juridique, fiscal, agricole, etc.

La plateforme intègre des applications métiers (PME Copilot, CRM IA, Fiscal Assistant, Legal Assistant) et offre une API REST ainsi que des SDK pour les développeurs.

## Concepts clés
- **Chat unifié** : dialogue avec plusieurs modèles IA dans une seule interface.
- **Reprompt multi‑modèles** : relance et comparaison de réponses entre modèles.
- **Assistants IAFactory** : assistants spécialisés pour chaque secteur ou métier.
- **Applications métiers** : outils IA pour PME, fiscalité, juridique, CRM, etc.
- **API & SDK** : intégration IAFactory dans vos propres applications.
