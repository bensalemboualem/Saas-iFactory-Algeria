Installing IAFactory Algeria

Complete guide to install and configure IAFactory Algeria on your system.

Web Application (No Installation Required)

The easiest way to use IAFactory Algeria is through our web application.

Instant Access

Go to app.iafactory-algeria.com

  • No installation required
  • Compatible with all browsers (Chrome, Firefox, Safari, Edge)
  • Accessible from PC, Mac, tablet, mobile
  • Automatic updates

Desktop Application (Windows/Mac/Linux)

Windows

  1. Download IAFactory-Algeria-Setup.exe
  2. Double-click the file
  3. Follow the installation wizard
  4. Launch IAFactory Algeria from the Start menu
Minimum Windows Requirements:
  • Windows 10 or higher (64-bit)
  • 4 GB RAM (8 GB recommended)
  • 500 MB disk space
  • Internet connection

macOS

  1. Download IAFactory-Algeria.dmg
  2. Open the DMG file
  3. Drag IAFactory to Applications
  4. Launch from Launchpad
macOS Gatekeeper:

If you see "Unverified Application":

  1. Right-click → Open
  2. Or: System Preferences → Security → "Open Anyway"

Linux

Debian/Ubuntu:

wget https://downloads.iafactory-algeria.com/iafactory-algeria.deb
sudo dpkg -i iafactory-algeria.deb
sudo apt-get install -f

Fedora/RHEL:

wget https://downloads.iafactory-algeria.com/iafactory-algeria.rpm
sudo rpm -i iafactory-algeria.rpm

AppImage (all distributions):

wget https://downloads.iafactory-algeria.com/IAFactory-Algeria.AppImage
chmod +x IAFactory-Algeria.AppImage
./IAFactory-Algeria.AppImage

Docker Installation

For developers and server deployments:

Docker Compose (Recommended)

# docker-compose.yml
version: '3.8'

services:
  iafactory:
    image: iafactory-algeria/platform:latest
    ports:
      - "8080:8080"
    environment:
      - OPENAI_API_KEY=${OPENAI_API_KEY}
      - ANTHROPIC_API_KEY=${ANTHROPIC_API_KEY}
    volumes:
      - ./data:/app/data
    restart: unless-stopped

Launch:

docker-compose up -d

Simple Docker

docker run -d \
  -p 8080:8080 \
  -e OPENAI_API_KEY=your_key \
  -v $(pwd)/data:/app/data \
  --name iafactory \
  iafactory-algeria/platform:latest

CLI Installation

For power users and automation:

Via npm

npm install -g @iafactory-algeria/cli

Via pip

pip install iafactory-algeria

Verify Installation

iafactory --version
iafactory auth login

SDK/API Installation

Python

pip install iafactory-algeria-sdk

Usage Example:

from iafactory import IAFactory

client = IAFactory(api_key="your_api_key")

response = client.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)

JavaScript/TypeScript

npm install @iafactory-algeria/sdk

Example:

import { IAFactory } from '@iafactory-algeria/sdk';

const client = new IAFactory({
  apiKey: 'your_api_key'
});

const response = await client.chat.completions.create({
  model: 'gpt-4',
  messages: [{ role: 'user', content: 'Hello!' }]
});

console.log(response.choices[0].message.content);

Initial Configuration

First Login

  1. Create an account
    • Email + password
    • Or Google/GitHub login
  2. Choose your plan
    • Starter (free)
    • Pro ($49/month)
    • Enterprise (custom quote)
  3. Configure your API keys (optional)
    • OpenAI, Anthropic, Google...
    • Or use IAFactory keys (included)

Configuration File

Create ~/.iafactory/config.yaml:

# IAFactory Algeria Configuration
api:
  url: https://api.iafactory-algeria.com/v1
  key: your_api_key_here

defaults:
  model: gpt-4
  temperature: 0.7
  max_tokens: 2048

providers:
  openai:
    api_key: sk-...
  anthropic:
    api_key: sk-ant-...

cache:
  enabled: true
  ttl: 3600

logging:
  level: info
  file: ~/.iafactory/logs/app.log

Troubleshooting

Problem: "Unable to connect"

Solutions:

  1. Check your Internet connection
  2. Temporarily disable proxy/VPN
  3. Check firewall (allow port 443)
  4. Test: ping api.iafactory-algeria.com

Problem: "Invalid API key"

Solutions:

  1. Verify the key is correctly copied (no spaces)
  2. Regenerate a new key from the dashboard
  3. Verify your subscription is active

Problem: "Installation blocked by antivirus"

Solutions:

  1. Add IAFactory to your antivirus whitelist
  2. Download from the official website only
  3. Verify the file's digital signature
