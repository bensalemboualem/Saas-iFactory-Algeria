Prompt Techniques

1. Be Specific

Bad: 
Write something about Python
Good: 
Write a Python function that:
- Takes a list of integers
- Returns the second largest number
- Handles edge cases (empty list, duplicates)
- Includes docstring and type hints

2. Provide Context

I'm building a React e-commerce app with TypeScript.
I'm using Zustand for state management.
I need a shopping cart component that:
- Shows items with quantity controls
- Calculates total with tax (10%)
- Persists to localStorage

3. Use Examples (Few-Shot)

Convert these texts to JSON:

Example 1:
Input: "John Doe, 25 years old, lives in NYC"
Output: {"name": "John Doe", "age": 25, "city": "NYC"}

Example 2:
Input: "Jane Smith from LA, age 30"
Output: {"name": "Jane Smith", "age": 30, "city": "LA"}

Now convert:
Input: "Ahmed Ali, 28, based in Algiers"

4. Chain of Thought

Solve this step by step:

Problem: Calculate the monthly cost for a team of 5 developers
- Base rate: $500/developer
- 20% discount for teams > 3
- Additional $100 for premium support
- 10% tax

Think through each step before giving the final answer.

5. Role Assignment

You are a senior security auditor with 15 years of experience.
Review this code for security vulnerabilities:

[paste code]

Focus on:
- SQL injection
- XSS vulnerabilities
- Authentication flaws
- Rate limiting issues

API Optimization

Reduce Costs

# Use mini models for simple tasks
response = client.chat.completions.create(
    model="gpt-4.1-mini",  # Cheaper, still good
    messages=[...],
    max_tokens=500  # Limit output
)

# Enable caching for repeated queries
response = client.chat.completions.create(
    model="gpt-4.1",
    messages=[...],
    cache=True  # Reuse cached responses
)

Improve Speed

# Use streaming for faster perceived response
for chunk in client.chat.completions.create(
    model="gpt-4.1-mini",
    messages=[...],
    stream=True
):
    print(chunk.choices[0].delta.content, end="")

# Batch requests when possible
responses = client.batch.create(
    requests=[
        {"model": "gpt-4.1-mini", "messages": [...]},
        {"model": "gpt-4.1-mini", "messages": [...]},
    ]
)

Common Mistakes to Avoid

1. Vague Prompts

Bad: "Help me with my code"
Bad: "It doesn't work"
Bad: "Make it better"

Good: "Fix the TypeError on line 15 of this Python function: [code]"

2. Overloading Single Requests

Bad: "Build me a complete e-commerce site with React, Node, MongoDB,
      authentication, payment processing, admin dashboard, and deploy it"

Good: Break into steps:
1. "Design the database schema for an e-commerce site"
2. "Create the Express API routes for products"
3. "Build the React product listing component"
...

3. Ignoring Context Limits

# Bad: Sending entire codebase
messages = [{"role": "user", "content": entire_project_code}]

# Good: Send only relevant files
messages = [
    {"role": "user", "content": f"""
    I have a bug in my checkout function.

    Relevant file (checkout.py):
    {checkout_code}

    Error: {error_message}
    """}
]

Security Best Practices

API Key Safety

  • Never commit API keys to version control
  • Use environment variables
  • Rotate keys regularly
  • Use different keys for dev/prod
# .env file (add to .gitignore!)
IAFACTORY_API_KEY=sk_live_your_key_here

# In code
import os
api_key = os.environ.get("IAFACTORY_API_KEY")

Data Privacy

  • Don't send sensitive personal data
  • Anonymize data before sending
  • Use local models (Ollama) for confidential data
  • Review IAFactory's data retention policies
