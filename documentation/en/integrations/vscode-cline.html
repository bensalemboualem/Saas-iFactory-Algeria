VS Code / Cline Integration

Use IAFactory Algeria directly in your favorite IDE with Cline (formerly Claude Dev).

Introduction

Cline is a powerful VS Code extension that transforms your editor into an AI development assistant. With IAFactory Algeria, you get access to all our models directly in your development workflow.

Benefits:
  • Real-time AI-assisted coding
  • Contextual code generation
  • Automatic refactoring
  • Auto-generated documentation
  • Automatic unit test creation

Installation

Prerequisites

  • VS Code (version 1.80+)
  • Node.js 16+ (for some features)
  • IAFactory Algeria API key

Step 1: Install the Extension

  1. Open VS Code
  2. Go to Extensions (Ctrl+Shift+X)
  3. Search for "Cline"
  4. Click "Install"

Or via command line:

code --install-extension saoudrizwan.claude-dev

Step 2: IAFactory Configuration

  1. Open Cline settings (Ctrl+Shift+P → "Cline: Settings")
  2. Select "OpenAI Compatible"
  3. Configure the settings:
{
  "cline.apiProvider": "openai-compatible",
  "cline.apiUrl": "https://api.iafactory-algeria.com/v1",
  "cline.apiKey": "YOUR_API_KEY",
  "cline.model": "gpt-4.1"
}

Step 3: Environment Variables (Recommended)

For better security, use a .env file:

# .env
IAFACTORY_API_KEY=sk_live_your_key
IAFACTORY_API_URL=https://api.iafactory-algeria.com/v1

Then in settings.json:

{
  "cline.apiKey": "${env:IAFACTORY_API_KEY}",
  "cline.apiUrl": "${env:IAFACTORY_API_URL}"
}

Advanced Configuration

Model Selection

Configure different models based on your needs:

{
  "cline.models": {
    "default": "gpt-4.1",
    "code": "codestral-2508",
    "chat": "gpt-4.1-mini",
    "analysis": "claude-sonnet-4-5"
  }
}

Context Window

{
  "cline.maxTokens": 8000,
  "cline.temperature": 0.2,
  "cline.contextLines": 50
}

Custom Instructions

{
  "cline.systemPrompt": "You are a Python expert specializing in Django. Write clean, tested, and documented code. Follow PEP 8 conventions."
}

Usage

Generate Code

  1. Open the command palette (Ctrl+Shift+P)
  2. Type "Cline: Generate Code"
  3. Describe what you want

Example:

Create a Python User class with:
- Attributes: id, name, email, created_at
- to_dict() method
- Email validation with regex
- Complete docstrings

Refactoring

  1. Select code to refactor
  2. Ctrl+Shift+P → "Cline: Refactor"
  3. Specify desired improvements

Generate Documentation

Select function → Cline: Generate Docstring

Generate Tests

Cline: Generate Tests for Current File

Troubleshooting

Connection Error

Error: "Unable to connect to API"

Solutions:

  1. Verify your API key
  2. Test the URL: curl https://api.iafactory-algeria.com/v1/models
  3. Check your firewall/proxy
  4. Confirm your subscription is active

Slow Responses

{
  "cline.timeout": 30000,
  "cline.maxTokens": 4000,
  "cline.model": "gpt-4.1-mini"  // Faster
}

Best Practices

Effective Prompts

Good 
Create a Python function that:
- Takes a list of numbers
- Returns the median
- Handles even/odd cases
- Type hints + docstring
- Unit tests included
Bad 
make a median function

Security

  • NEVER commit your API key
  • Use .env + .gitignore
  • Rotate keys regularly
  • Use different keys for dev/prod
