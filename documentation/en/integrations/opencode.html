üì¶ Installation

Via Marketplace

  1. Open VS Code
  2. Go to Extensions (Ctrl+Shift+X)
  3. Search for IAFactory Algeria
  4. Click Install

Via CLI

code --install-extension iafactory-algeria.vscode-iafactory

‚öôÔ∏è Configuration

Configure your API Key

  1. Open Settings (Ctrl+,)
  2. Search for IAFactory
  3. Paste your API key in IAFactory: API Key

Or via settings.json:

{
  "iafactory.apiKey": "YOUR_API_KEY",
  "iafactory.defaultModel": "gpt-4.1",
  "iafactory.autoComplete": true,
  "iafactory.temperature": 0.7
}

Available Settings

Parameter Description Default
apiKey Your IAFactory API key -
defaultModel Default model gpt-4.1
autoComplete Auto-completion true
temperature Creativity (0-1) 0.7
maxTokens Maximum tokens 2000

üöÄ Features

1. Inline Completion

Start typing and AI automatically suggests the continuation:

// Just type:
function calculateFibonacci

// AI completes:
function calculateFibonacci(n) {
    if (n <= 1) return n;
    return calculateFibonacci(n - 1) + calculateFibonacci(n - 2);
}

Shortcuts:

  • Tab: Accept suggestion
  • Esc: Reject
  • Alt+]: Next suggestion

2. Chat Sidebar

Open AI chat with Ctrl+Shift+I or via the sidebar icon.

Quick commands:

  • /explain: Explain selected code
  • /doc: Generate documentation
  • /test: Generate tests
  • /refactor: Refactoring suggestions
  • /fix: Fix errors

3. Context Menu

Right-click on code ‚Üí IAFactory:

  • Explain Code: Detailed explanation
  • Generate Tests: Automatic unit tests
  • Add Documentation: JSDoc/PyDoc/etc.
  • Optimize Code: Optimization suggestions
  • Translate Comments: FR/EN/AR translation

4. Code Actions

Lightbulb (üí°) on errors/warnings:

  • Fix with AI: Automatic fix
  • Explain Error: Error explanation
  • Suggest Alternative: Alternative approaches

üéØ Usage Examples

Test Generation

Select a function ‚Üí Right-click ‚Üí IAFactory: Generate Tests

# Your function
def calculate_discount(price, percentage):
    if percentage < 0 or percentage > 100:
        raise ValueError("Percentage must be between 0 and 100")
    return price * (1 - percentage / 100)

# Auto-generated tests
import pytest

def test_calculate_discount_normal():
    assert calculate_discount(100, 10) == 90
    assert calculate_discount(50, 20) == 40

def test_calculate_discount_zero():
    assert calculate_discount(100, 0) == 100

def test_calculate_discount_full():
    assert calculate_discount(100, 100) == 0

def test_calculate_discount_invalid():
    with pytest.raises(ValueError):
        calculate_discount(100, -10)
    with pytest.raises(ValueError):
        calculate_discount(100, 150)
