n8n Integration

Automate your workflows with n8n and IAFactory Algeria API to create powerful automation scenarios.

üîÑ What is n8n?

n8n is an open-source automation platform that lets you connect different services and automate your tasks. With IAFactory Algeria, you can integrate AI models directly into your workflows.

üí° Use Cases:
  • Automatic content generation from forms
  • Incoming data analysis with AI
  • Automatic document translation
  • Image generation for marketing campaigns
  • Automatic email or document summarization

Self-Hosted Version

  --name n8n \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \

npm install -g n8n

Cloud Version

  1. Sign up on n8n.io
  2. Create your first workflow
  3. Add IAFactory Algeria credentials

‚öôÔ∏è Configuration

Add Your IAFactory Credentials

  1. In n8n, go to Settings ‚Üí Credentials
  2. Click Add Credentials
  3. Select HTTP Request
  4. Configure:
    • Name: IAFactory Algeria API
    • Authentication: Header Auth
    • Name: Authorization
    • Value: Bearer YOUR_API_KEY

üöÄ First Workflows

Workflow 1: Content Generation from Form

Goal: Receive a web form and automatically generate marketing content with AI.

Workflow Setup:

  1. Webhook Trigger: Receive form data
  2. HTTP Request: Call IAFactory API 
    {
  "method": "POST",
  "url": "https://api.iafactory-algeria.com/v1/chat/completions",
  "authentication": "IAFactory Algeria API",
  "body": {
    "model": "gpt-4.1",
    "messages": [
      {
        "role": "system",
        "content": "You are an expert copywriter. Generate persuasive marketing text."
      },
      {
        "role": "user",
        "content": "Product: {{$json.product_name}}\nDescription: {{$json.description}}"
      }
    ],
    "temperature": 0.7,
    "max_tokens": 500
  }
}
  3. Set Node: Extract generated content
  4. Email / Slack / Database: Send result

Workflow 2: Email Sentiment Analysis

Goal: Automatically analyze incoming email sentiment and categorize.

{
  "nodes": [
    {
      "name": "Email Trigger",
      "type": "n8n-nodes-base.emailTrigger"
    },
    {
      "name": "IAFactory Analysis",
      "type": "n8n-nodes-base.httpRequest",
      "parameters": {
        "url": "https://api.iafactory-algeria.com/v1/chat/completions",
        "method": "POST",
        "body": {
          "model": "gpt-4.1-mini",
          "messages": [
            {
              "role": "system",
              "content": "Analyze email sentiment. Reply only: POSITIVE, NEGATIVE or NEUTRAL"
            },
            {
              "role": "user",
              "content": "{{$json.body}}"
            }
          ],
          "temperature": 0.1
        }
      }
    },
    {
      "name": "Route by Sentiment",
      "type": "n8n-nodes-base.switch"
    },
    {
      "name": "Alert Team",
      "type": "n8n-nodes-base.slack"
    }
  ]
}

‚ú® Best Practices

1. Error Handling

  • Always add an Error Trigger node
  • Use Retry on Fail for API calls
  • Log errors to a database
  • Configure Slack/Email alerts

2. Cost Optimization

  • Use gpt-4.1-mini for simple tasks
  • Cache frequent responses
  • Limit max_tokens as needed
  • Use conditions to avoid unnecessary calls
