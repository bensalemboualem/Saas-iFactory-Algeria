‚úçÔ∏è Writing Effective Prompts

Structure of a Good Prompt

[SUBJECT] + [STYLE] + [DETAILS] + [QUALITY] + [TECHNICAL PARAMETERS]

Example:
"Humanoid robot, cyberpunk style, blue and pink neons,
futuristic city in background, ultra detailed, 8K,
cinematic lighting, octane render"

Standard Formulas

Professional Portrait

"Professional portrait of a [profession], [age] years old,
neutral background, studio lighting, high resolution,
photographed with Canon EOS R5, 85mm f/1.4 lens"

Artistic Landscape

"Landscape of [location], [season], [time of day],
style [artist or artistic movement],
colors [palette], composition [composition rule],
natural light, ultra detailed"

Product Illustration

"Product photo of [product], minimalist white background,
professional studio lighting, soft shadows,
ultra high definition, e-commerce catalog style"

Concept Art

"Concept art of [subject], style [game or film],
color palette [colors], perspective [angle],
intricate details, dramatic lighting,
artstation trending, digital painting"

Create Variations

def generate_variations(base_prompt, variations):
    """Generate multiple variations of the same concept"""
    results = []
    
    for style in variations:
        prompt = f"{base_prompt}, style {style}"
        
        response = requests.post(url, headers=headers, json={
            "model": "dall-e-3",
            "prompt": prompt,
            "size": "1024x1024"
        })
        
        results.append({
            'style': style,
            'url': response.json()['data'][0]['url']
        })
    
    return results

# Usage
base = "Traditional Algerian caf√©"
styles = ["watercolor", "photorealistic", "art deco", "minimalist"]

variations = generate_variations(base, styles)

Batch Generation

import asyncio
import aiohttp

async def generate_image_async(session, prompt):
    """Generate an image asynchronously"""
    async with session.post(url, headers=headers, json={
        "model": "stable-diffusion-xl",
        "prompt": prompt
    }) as response:
        data = await response.json()
        return data['data'][0]['url']

async def batch_generate(prompts):
    """Generate multiple images in parallel"""
    async with aiohttp.ClientSession() as session:
        tasks = [generate_image_async(session, p) for p in prompts]
        return await asyncio.gather(*tasks)

# Usage
prompts = [
    "Modern logo for tech startup",
    "Minimalist icon for mobile app",
    "Web banner for e-commerce"
]

images = asyncio.run(batch_generate(prompts))

‚ú® Best Practices

Negative Prompts

Specify what you DON'T want in the image:

{
  "prompt": "Professional portrait, studio lighting",
  "negative_prompt": "blur, distortion, low quality, watermark, text, deformed hands"
}

Progressive Iteration

  1. Basic prompt: "A cat"
  2. Add style: "A cat, watercolor style"
  3. Add details: "Black and white cat, watercolor style, floral background"
  4. Add technical: "Black and white cat, watercolor style, floral background, soft lighting, 4K"

Use References

Style of [famous artist]
Like [well-known work]
In the manner of [artistic movement]
Inspired by [film/game]

Examples:
- "Studio Ghibli style"
- "like Blade Runner 2049"
- "in the manner of Van Gogh"
- "inspired by Cyberpunk 2077"

üîß Troubleshooting

Blurry or Low Quality Image

Solutions:

  • Add "ultra detailed, 8K, sharp focus"
  • Use quality="hd"
  • Increase resolution
  • Try a different model (Flux Pro)

Style Not Respected

Solutions:

  • Be more specific about style
  • Add artistic references
  • Use negative prompt
  • Try Midjourney for artistic styles

Illegible Text in Image

Solutions:

  • Use DALL-E 3 (better for text)
  • Specify "clear readable text"
  • Simplify requested text
  • Generate text separately then compose
