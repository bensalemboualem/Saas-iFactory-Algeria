Writing Prompts

Master the art of communicating with AI to get the best results.

Fundamentals

1. Clarity

Good 
Write a Python function that calculates the average of a list of numbers.
Handle empty lists by returning 0.
Bad 
do something to calculate

2. Context

I'm a backend developer using Node.js.
I'm working on a REST API for an e-commerce site.
I need a middleware function to validate JWT tokens.

Constraints:
- Use jsonwebtoken v9
- Handle expiration
- Log errors
- TypeScript

3. Specificity

Vague Specific
"Help me with SQL" "Write a SQL query to get the top 10 customers by spending in 2024"
"Explain React" "Explain the hooks concept in React with a useState and useEffect example"
"Debug my code" "This Python code returns KeyError on line 15. Here's the code: [code]. How to fix?"

Prompt Formulas

Formula: Role + Task + Format

You are a [ROLE].

Task: [WHAT YOU WANT]

Output format: [HOW YOU WANT IT]
Concrete Example: 
You are a cybersecurity expert.

Task: Analyze this PHP code and identify security vulnerabilities.

Output format:
- Numbered list of vulnerabilities
- Criticality level (Low/Medium/Critical)
- Recommended solution for each

Formula: Context + Action + Constraints

Context: [SITUATION]

Action: [WHAT NEEDS TO BE DONE]

Constraints:
- [CONSTRAINT 1]
- [CONSTRAINT 2]
- [CONSTRAINT 3]
Example: 
Context: Mobile fitness app in React Native

Action: Create a Timer component for exercises

Constraints:
- Use React hooks
- Display format MM:SS
- Play/pause/reset buttons
- Sound notification at end
- Material Design style

Formula: Few-Shot Learning

Here are examples:

Example 1:
Input: [example 1 input]
Output: [example 1 output]

Example 2:
Input: [example 2 input]
Output: [example 2 output]

Now do the same for:
Input: [your input]

Advanced Techniques

Chain-of-Thought

Solve this problem step by step:

Problem: Calculate the total cost of a project

Data:
- 3 developers at $500/day
- 2 designers at $400/day
- 20 working days
- +20% overhead

Think through each step before answering.

Role-Playing

You are a senior code reviewer with 15 years of experience.
You are strict about:
- Code readability
- Performance
- Security
- Best practices

Review this code as you would for a junior on your team:

[code here]

Constraint-Based

Generate 5 brand names for a food delivery startup.

MANDATORY constraints:
- Maximum 8 letters
- Easy to pronounce in English and Arabic
- Available as .com domain
- Evokes speed or freshness
- No numbers

For each name, explain why it meets the constraints.

Prompt Optimization

Temperature Usage

{
  "model": "gpt-4.1",
  "messages": [...],
  "temperature": 0.2  // For code/extraction
}

{
  "model": "gpt-4.1",
  "messages": [...],
  "temperature": 1.3  // For brainstorming/creativity
}

Token Optimization

Verbose (expensive) 
Could you please help me understand how I might potentially solve this complex problem...
Concise (economical) 
How to solve: [specific problem]
Code: [snippet]
Error: [error message]

Anti-Patterns to Avoid

1. Too Vague

"Help me"
"It doesn't work"
"Do something with this code"

2. Too Complex at Once

Bad: "Create a complete app with React frontend, Node backend, MongoDB..."

Good: Break into steps:
1. "Structure Node.js backend with Express"
2. "Add JWT auth system"
3. etc...

Ready-to-Use Templates

Technical Documentation

Document this function/class:

Code:
[code]

Generate:
1. Description of what it does
2. Parameters (type, description)
3. Return value
4. Possible exceptions
5. Usage example
6. Technical notes

Format: Python Docstring (Google Style)

API Design

Design a REST API for [feature].

Specifications:
- Resources: [list]
- CRUD actions needed
- Authentication: JWT
- Pagination required
- Rate limiting: 100 req/min

Provide:
1. Endpoints with HTTP methods
2. Request/response JSON structure
3. HTTP status codes
4. cURL example for each endpoint

Evaluating Prompt Quality

Checklist

  • Clear objective?
  • Context provided?
  • Constraints specified?
  • Output format defined?
  • Examples if needed?
  • Appropriate detail level?
  • Precise vocabulary?
