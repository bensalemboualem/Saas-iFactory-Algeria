تثبيت IAFactory Algeria

دليل كامل لتثبيت وتكوين IAFactory Algeria على نظامك.

تطبيق الويب (بدون تثبيت)

أسهل طريقة لاستخدام IAFactory Algeria هي عبر تطبيق الويب الخاص بنا.

وصول فوري

اذهب إلى app.iafactory-algeria.com

  • لا يتطلب تثبيت
  • متوافق مع جميع المتصفحات (Chrome، Firefox، Safari، Edge)
  • متاح من الكمبيوتر، Mac، الجهاز اللوحي، الهاتف المحمول
  • تحديثات تلقائية

تطبيق سطح المكتب (Windows/Mac/Linux)

Windows

  1. قم بتحميل IAFactory-Algeria-Setup.exe
  2. انقر نقرًا مزدوجًا على الملف
  3. اتبع معالج التثبيت
  4. شغّل IAFactory Algeria من قائمة ابدأ
الحد الأدنى لمتطلبات Windows:
  • Windows 10 أو أحدث (64-bit)
  • 4 جيجابايت رام (يُوصى بـ 8 جيجابايت)
  • 500 ميجابايت مساحة قرص
  • اتصال بالإنترنت

macOS

  1. قم بتحميل IAFactory-Algeria.dmg
  2. افتح ملف DMG
  3. اسحب IAFactory إلى التطبيقات
  4. شغّل من Launchpad
macOS Gatekeeper:

إذا رأيت "تطبيق غير موثق":

  1. انقر بزر الماوس الأيمن → فتح
  2. أو: تفضيلات النظام → الأمان → "فتح على أي حال"

Linux

Debian/Ubuntu:

wget https://downloads.iafactory-algeria.com/iafactory-algeria.deb
sudo dpkg -i iafactory-algeria.deb
sudo apt-get install -f

Fedora/RHEL:

wget https://downloads.iafactory-algeria.com/iafactory-algeria.rpm
sudo rpm -i iafactory-algeria.rpm

AppImage (جميع التوزيعات):

wget https://downloads.iafactory-algeria.com/IAFactory-Algeria.AppImage
chmod +x IAFactory-Algeria.AppImage
./IAFactory-Algeria.AppImage

iOS

  1. افتح App Store

  3. اضغط على "تحميل"

تثبيت Docker

للمطورين والنشر على الخوادم:

Docker Compose (موصى به)

# docker-compose.yml
version: '3.8'

services:
  iafactory:
    image: iafactory-algeria/platform:latest
    ports:
      - "8080:8080"
    environment:
      - OPENAI_API_KEY=${OPENAI_API_KEY}
      - ANTHROPIC_API_KEY=${ANTHROPIC_API_KEY}
    volumes:
      - ./data:/app/data
    restart: unless-stopped

تشغيل:

docker-compose up -d

Docker بسيط

docker run -d \
  -p 8080:8080 \
  -e OPENAI_API_KEY=your_key \
  -v $(pwd)/data:/app/data \
  --name iafactory \
  iafactory-algeria/platform:latest

تثبيت CLI

للمستخدمين المتقدمين والأتمتة:

عبر npm

npm install -g @iafactory-algeria/cli

عبر pip

pip install iafactory-algeria

التحقق من التثبيت

iafactory --version
iafactory auth login

تثبيت SDK/API

Python

pip install iafactory-algeria-sdk

مثال على الاستخدام:

from iafactory import IAFactory

client = IAFactory(api_key="your_api_key")

response = client.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "مرحبا!"}]
)

print(response.choices[0].message.content)

JavaScript/TypeScript

npm install @iafactory-algeria/sdk

مثال:

import { IAFactory } from '@iafactory-algeria/sdk';

const client = new IAFactory({
  apiKey: 'your_api_key'
});

const response = await client.chat.completions.create({
  model: 'gpt-4',
  messages: [{ role: 'user', content: 'مرحبا!' }]
});

console.log(response.choices[0].message.content);

التكوين الأولي

تسجيل الدخول الأول

  1. إنشاء حساب
    • البريد الإلكتروني + كلمة المرور
    • أو تسجيل الدخول عبر Google/GitHub
  2. اختيار خطتك
    • Starter (مجاني)
    • Pro (49$/شهر)
    • Enterprise (عرض مخصص)
  3. تكوين مفاتيح API (اختياري)
    • OpenAI، Anthropic، Google...
    • أو استخدام مفاتيح IAFactory (مضمنة)

ملف التكوين

أنشئ ~/.iafactory/config.yaml:

# تكوين IAFactory Algeria
api:
  url: https://api.iafactory-algeria.com/v1
  key: your_api_key_here

defaults:
  model: gpt-4
  temperature: 0.7
  max_tokens: 2048

providers:
  openai:
    api_key: sk-...
  anthropic:
    api_key: sk-ant-...

cache:
  enabled: true
  ttl: 3600

logging:
  level: info
  file: ~/.iafactory/logs/app.log

استكشاف الأخطاء

مشكلة: "غير قادر على الاتصال"

الحلول:

  1. تحقق من اتصالك بالإنترنت
  2. عطّل البروكسي/VPN مؤقتًا
  3. تحقق من جدار الحماية (السماح بالمنفذ 443)
  4. اختبر: ping api.iafactory-algeria.com

مشكلة: "مفتاح API غير صالح"

الحلول:

  1. تحقق من نسخ المفتاح بشكل صحيح (بدون مسافات)
  2. أنشئ مفتاحًا جديدًا من لوحة التحكم
  3. تحقق من أن اشتراكك نشط

مشكلة: "التثبيت محظور من مضاد الفيروسات"

الحلول:

  1. أضف IAFactory إلى القائمة البيضاء لمضاد الفيروسات
  2. قم بالتحميل من الموقع الرسمي فقط
  3. تحقق من التوقيع الرقمي للملف
