# إدارة مفاتيح API

تأمين مفاتيح API أمر أساسي لحماية الوصول إلى خدمات IAFactory الخاصة بك.

## إنشاء مفتاح API
- سجّل الدخول إلى حساب المستخدم الخاص بك في IAFactory
- انتقل إلى قسم "مفاتيح API"
- اضغط على "إنشاء مفتاح جديد"
- انسخ المفتاح واحتفظ به في مكان آمن

## أفضل الممارسات
- لا تشارك مفتاح API الخاص بك علنًا أبدًا
- لا تخزن المفتاح في مستودع git عام أو في كود مصدر متاح
- استخدم متغيرات البيئة لإدخال المفتاح على الخادم
- قم بإلغاء أي مفتاح تم اختراقه فورًا

## تدوير المفاتيح
- غيّر مفاتيح API الخاصة بك بانتظام
- احذف المفاتيح القديمة غير المستخدمة

## مثال على الاستخدام
```bash
export API_KEY=مفتاحك
curl -H "Authorization: Bearer $API_KEY" https://api.iafactory.dz/v1/endpoint
```

لأي استفسار أو مشكلة أمنية، راسل support@iafactory.dz
