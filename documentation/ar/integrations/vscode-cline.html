تكامل VS Code / Cline

استخدم IAFactory Algeria مباشرة في IDE المفضل لديك مع Cline (سابقًا Claude Dev).

مقدمة

Cline هو إضافة VS Code قوية تحول محررك إلى مساعد تطوير بالذكاء الاصطناعي. مع IAFactory Algeria، تحصل على وصول إلى جميع نماذجنا مباشرة في سير عمل التطوير الخاص بك.

المزايا:
  • مساعدة الكود بالذكاء الاصطناعي في الوقت الحقيقي
  • توليد الكود السياقي
  • إعادة الهيكلة التلقائية
  • التوثيق المولد تلقائيًا
  • إنشاء اختبارات الوحدة تلقائيًا

التثبيت

المتطلبات الأساسية

  • VS Code (الإصدار 1.80+)
  • Node.js 16+ (لبعض الميزات)
  • مفتاح API من IAFactory Algeria

الخطوة 1: تثبيت الإضافة

  1. افتح VS Code
  2. اذهب إلى الإضافات (Ctrl+Shift+X)
  3. ابحث عن "Cline"
  4. انقر على "تثبيت"

أو عبر سطر الأوامر:

code --install-extension saoudrizwan.claude-dev

الخطوة 2: تكوين IAFactory

  1. افتح إعدادات Cline (Ctrl+Shift+P → "Cline: Settings")
  2. اختر "OpenAI Compatible"
  3. كوّن الإعدادات:
{
  "cline.apiProvider": "openai-compatible",
  "cline.apiUrl": "https://api.iafactory-algeria.com/v1",
  "cline.apiKey": "YOUR_API_KEY",
  "cline.model": "gpt-4.1"
}

الخطوة 3: متغيرات البيئة (موصى به)

لأمان أفضل، استخدم ملف .env:

# .env
IAFACTORY_API_KEY=sk_live_your_key
IAFACTORY_API_URL=https://api.iafactory-algeria.com/v1

ثم في settings.json:

{
  "cline.apiKey": "${env:IAFACTORY_API_KEY}",
  "cline.apiUrl": "${env:IAFACTORY_API_URL}"
}

التكوين المتقدم

اختيار النموذج

كوّن نماذج مختلفة حسب احتياجاتك:

{
  "cline.models": {
    "default": "gpt-4.1",
    "code": "codestral-2508",
    "chat": "gpt-4.1-mini",
    "analysis": "claude-sonnet-4-5"
  }
}

نافذة السياق

{
  "cline.maxTokens": 8000,
  "cline.temperature": 0.2,
  "cline.contextLines": 50
}

تعليمات مخصصة

{
  "cline.systemPrompt": "أنت خبير Python متخصص في Django. اكتب كودًا نظيفًا ومُختبرًا وموثقًا. اتبع اتفاقيات PEP 8."
}

الاستخدام

توليد الكود

  1. افتح لوحة الأوامر (Ctrl+Shift+P)
  2. اكتب "Cline: Generate Code"
  3. صِف ما تريده

مثال:

أنشئ فئة User في Python مع:
- السمات: id، name، email، created_at
- طريقة to_dict()
- التحقق من البريد الإلكتروني باستخدام regex
- docstrings كاملة

إعادة الهيكلة

  1. حدد الكود المراد إعادة هيكلته
  2. Ctrl+Shift+P → "Cline: Refactor"
  3. حدد التحسينات المطلوبة

توليد التوثيق

حدد الدالة → Cline: Generate Docstring

توليد الاختبارات

Cline: Generate Tests for Current File

اختصارات لوحة المفاتيح

الاختصار الإجراء
Ctrl+Shift+Lفتح محادثة Cline
Ctrl+Shift+Gتوليد الكود
Ctrl+Shift+Rإعادة هيكلة التحديد
Ctrl+Shift+Dتوليد docstring
Ctrl+Shift+Tتوليد الاختبارات
Ctrl+Shift+Eشرح الكود

استكشاف الأخطاء

خطأ في الاتصال

خطأ: "Unable to connect to API"

الحلول:

  1. تحقق من مفتاح API الخاص بك
  2. اختبر الرابط: curl https://api.iafactory-algeria.com/v1/models
  3. تحقق من جدار الحماية/البروكسي
  4. تأكد من أن اشتراكك نشط

استجابات بطيئة

{
  "cline.timeout": 30000,
  "cline.maxTokens": 4000,
  "cline.model": "gpt-4.1-mini"  // أسرع
}
