الأساسيات

1. الوضوح

جيد 
اكتب دالة Python تحسب متوسط قائمة من الأرقام.
تعامل مع القوائم الفارغة بإرجاع 0.
سيء 
افعل شيئًا للحساب

2. السياق

أنا مطور backend أستخدم Node.js.
أعمل على REST API لموقع تجارة إلكترونية.
أحتاج دالة middleware للتحقق من رموز JWT.

القيود:
- استخدم jsonwebtoken v9
- تعامل مع انتهاء الصلاحية
- سجّل الأخطاء
- TypeScript

3. التحديد

غامض محدد
"ساعدني في SQL" "اكتب استعلام SQL للحصول على أفضل 10 عملاء حسب الإنفاق في 2024"
"اشرح React" "اشرح مفهوم hooks في React مع مثال useState و useEffect"
"صحح الكود" "هذا الكود Python يُرجع KeyError في السطر 15. إليك الكود: [الكود]. كيف أصلحه؟"
