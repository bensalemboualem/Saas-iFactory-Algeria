# Getting Started / البدء

## Créer un compte
Inscrivez-vous sur IAFactory Algeria et accédez à votre espace personnel.

## Première question dans le chat
Choisissez un assistant ou un modèle IA et posez votre première question.

## Utiliser le reprompt
Relancez une question avec un autre modèle pour comparer les réponses.

## Créer un premier assistant
Créez un assistant personnalisé (ex : Fiscalité PME).

## Pour les développeurs
Consultez la [API Quick Start](../api/quick-start.md).
