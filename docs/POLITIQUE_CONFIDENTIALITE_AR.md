# سياسة الخصوصية (الجزائر)

وفقًا للقانون رقم 18-07 المتعلق بحماية الأشخاص الطبيعيين في معالجة البيانات ذات الطابع الشخصي في الجزائر، نلتزم بحماية بياناتك.

**البيانات التي يتم جمعها:**
- البريد الإلكتروني، الاسم، معلومات الدخول
- سجلات تقنية للأمان والدعم

**الاستخدام:**
- إدارة الحساب
- دعم المستخدم
- تحسين الخدمة

**حقوق المستخدم:**
لديك الحق في الوصول إلى بياناتك وتصحيحها والاعتراض عليها وحذفها. لممارسة هذه الحقوق، راسل support@iafactory.dz

**مدة الاحتفاظ:**
يتم الاحتفاظ بالبيانات فقط للمدة اللازمة لغرض المعالجة.

**نقل البيانات:**
لا يتم نقل أي بيانات خارج الجزائر دون موافقة صريحة.

**الكوكيز:**
راجع سياسة الكوكيز.
